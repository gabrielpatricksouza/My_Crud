# My Crud

Aplicativo para criar e editar perfil.

## Tecnologia s

O que foi utilizado para criar o app:

- Flutter
- Dart
- Modular
- Mobx
- Firebase

## Funcionalidades
Todas as features do app:

- Login (email-senha)
- Cadastro (email - senha - foto - CPF - telefone - sexo - estado civil)
- Home: 
  - Visualiza perfil
  - Editar perfil (foto - nome - email - telefone)
  - LogOff do sistema
